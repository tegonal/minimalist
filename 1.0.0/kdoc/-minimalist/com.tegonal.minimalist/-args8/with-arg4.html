withArg4

abstract fun withArg4(value: A4, representation: String? = null): Args8<A1, A2, A3, A4, A5, A6, A7, A8>

Creates a new Args8 by coping this Args8 but replaces the argument 4 (Args8.a4) with the given value (and its representation with the given representation).

Return

The newly created Args8.

Since

1.0.0

Parameters

value

the new value to use for argument 4.

representation

the new representation to use for the argument 4 where null means let the algorithm determine a representation.
